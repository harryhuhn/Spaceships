Welcome to Spaceships, a simple 2-player game created using Pygame.

Before you start, ensure you have the following installed:
  - Python (version 3.6 or higher)
  - Pygame

Clone the repository:
  - git clone https://github.com/harryhuhn/Spaceships

Navigate to the project directory:
  - cd spaceships-game

Install dependencies:
  - pip install -r requirements.txt

How to Play:
  1. Run the game:
    - python main.py
  2. Controls:
    - Player 1 (Yellow Spaceship):
      - Move Up: W
      - Move Down: S
      - Move Left: A
      - Move Right: D
      - Shoot: Left Ctrl
    - Player 2 (Red Spaceship):
      - Move Up: Arrow Up
      - Move Down: Arrow Down
      - Move Left: Arrow Left
      - Move Right: Arrow Right
      - Shoot: Right Ctrl
  3. Objective:
    - Each player starts with 10 health points.
    - Shoot your opponent to reduce their health.
    - The first player to reduce their opponent's health to 0 wins.
